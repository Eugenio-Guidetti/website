PlatinaMI

Cos'è PlatinaMI

PlatinaMI è un'applicazione web che permette agli utenti di creare e completare obiettivi ispirati ai “trofei” dei videogiochi, ma applicati alla vita reale.

Come funziona

Ogni utente può creare i propri obiettivi personalizzati, organizzati in liste di obiettivi, all'interno di gruppi condivisi con altri utenti. I partecipanti possono:

  • Visualizzare e completare gli obiettivi del gruppo.
  • Controllare lo stato di avanzamento e le note degli altri partecipanti.

Per ogni obiettivo è inoltre possibile:

  • Contrassegnarlo come "Sbloccante": una volta completato, rende visibili tutti gli altri obiettivi della lista.
  • Modificare il "Progresso massimo", 1 di default, per creare obiettivi del tipo "fai qualcosa n volte".
  • Aggiungere note, visibili a tutti i partecipanti del gruppo che possono completare l'obiettivo.

Sviluppo del progetto

Il progetto è composto da:

  • Un database MySQL
  • Un back-end sviluppato in Java con il framework Spring Boot che espone un'API RESTful
  • Un front-end Android sviluppato in Java con Android Studio

L'applicazione Android è ancora in fase di sviluppo. È possibile partecipare al programma di testing qui.
